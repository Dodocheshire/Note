# 模型输出与预测标签

### 预测标签

假设图像分类问题中模型输出层为(n_classes, )，表示n_classes个类别得分(logits值)。

```python
# input_batch: (batch_size, C, H, W)
with torch.no_grad():
  output = model(input_batch)
# output: (batch_size, n_classes) logits
# 将logits转换为[0,1]之间的概率值,使用softmax函数得到: (batch_size, n_classes)
probabilities = torch.nn.functional.softmax(output, dim=1)

```

已知label_id对应的类别标签在`imagenet_classes.txt`​中，可以通过`torch.topk`查找预测概率最大的k个类别

```python
with open("imagenet_classes.txt", mode="r") as f:
  categories = [s.strip() for s in f.readlines()]

values, indices = torch.topk(probabilities, k=5, dim=1)
# 检查第1个样本最可能的k个类别和它们的概率值
for i in range(values[0].shape[0]):
  print(categories[indices[0][i], values[0][i].item())
```

- `torch.topk`​返回一个**二元组**​ **​`(values, indices)`​** 

  - **​`values`​**​：前k个最大（或最小）的**数值本身**
  - **​`indices`​**​：这些值在原始`input`​张量里、沿着`dim`​这个维度​**排第几位**（原始下标）

- 在这个例子中，`values`​形状`(batch_size, k)`​，`indices`​**形状**​`(batch_size, k)`

- 其他参数：`largest: bool = True`​, `sorted: bool = True` sorted保证了这些筛选出来的元素是按顺序排列的。
