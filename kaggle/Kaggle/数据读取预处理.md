# 数据读取预处理

### image -> input_tensor

#### 下载图片

- 使用urllib的request模块下载

```python
from urllib import request
url = "https://github.com/pytorch/hub/raw/master/images/dog.jpg"
local_path = "dog.jpg"
request.urlretrieve(url, local_path)
```

#### 读取图片

```python
from PIL import Image
# 如果是灰度图，转成RGB图
dog_img = Image.open(local_path).convert("RGB")
```

#### PIL.Image -> torch.Tensor

我们通常使用torchvision的**transforms**

- Compose

```python
from torchvision import transforms
eval_transform = transforms.Compose([
    transforms.Resize(int(IMAGE_SIZE * 1.14)),
    transforms.CenterCrop(IMAGE_SIZE), # 保证中心裁剪（得先调整比IMAGE_SIZE略大再裁剪才有效)
    transforms.ToTensor(),
    transforms.Normalize(mean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225]),
])
input_tensor = eval_transform(dog_img).unsqueeze(0)
```

- ToTensor:

  1. 把PIL图片（0~255的整数像素）从shape (H, W, C ) —> (C, H, W)
  2. 所有数除以255，将数据归一化到 [0,1]
- torch.unsqueeze(input, dim) 在原有tensor的axis=dim出新建一个axis，并将该处的维度设置为1.

#### Normalize

定义：把图片每个颜色通道的像素值减去均值、除以标准差。让数值分布落在一个统一的范围内。

**问题：** 3个通道的mean和std怎么计算？

**回答：** 在**整个ImageNet训练集**上，把所有图片的像素值缩放到`[0,1]`​范围后，**分别对R、G、B三个通道**统计出来的：

$$
\text{mean}c = \frac{1}{N \cdot H \cdot W}\sum{\text{所有图片},\text{所有像素}} \text{pixel}_c
$$
